# 📊 Retail Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes retail sales data using SQL, Python, and Power BI to uncover sales trends, customer behavior, product performance, and regional performance. The project demonstrates and end-to-end Data Analytics workflow from data cleaning to business recommendations.

## 🎯 Project Objectives

- Clean and prepare raw retail sales data.
- Perform SQL-based business analysis.
- Conduct Exploratory Data Analysis (EDA) using Python.
- Build an interactive Power BI dashboard.
- Generate business insights and actionable recommendations.

## 🛠️ Tools & Technologies

- Python ( Pandas, NumPy, Matplotlib)
- MySQL
- Power BI
- Jupyter Notebook
- Excel
- GitHub

## 💡 Key Business Insights

- Technology is the highest-selling category.
- The West region generated the highest revenue.
- Consumer is the largest customer segment.
- Sales peak during November.

## 📌 Business Recommendations

- Increase marketing in low-performing regions.
- Foucs inventoroy on high-performing products.
- Improve customer retention through loyalty programs.
